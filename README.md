# Онлайн-конструктор портфолио

Веб-приложение для создания персональных портфолио без навыков программирования.

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка базы данных

```bash
python manage.py migrate
python fix_db.py
```

### 3. Запуск сервера

```bash
python manage.py runserver
```

### 4. Откройте в браузере

- http://127.0.0.1:8000/auth/login/ - страница входа
- http://127.0.0.1:8000/ - главная страница (после входа)

## Структура проекта

- `accounts/` - управление пользователями и аутентификация
- `portfolio/` - создание и управление портфолио
- `admin_panel/` - административная панель
- `templates/` - HTML шаблоны

## Исправление ошибок

Если возникает ошибка с базой данных, запустите:

```bash
python fix_db.py
```

Или используйте скрипт:

```bash
ИСПРАВИТЬ_ОШИБКУ.bat
```
