# Быстрый старт

## Первый запуск

1. **Запустите `setup.bat`** - это создаст виртуальное окружение и установит все зависимости

2. **Запустите `run.bat`** - это запустит сервер разработки

3. Откройте браузер и перейдите на `http://127.0.0.1:8000`

## Важно!

**НЕ запускайте `manage.py` напрямую!** Всегда используйте:
- `manage.bat` для команд Django (например: `manage.bat migrate`, `manage.bat createsuperuser`)
- `run.bat` для запуска сервера
- Или активируйте виртуальное окружение вручную: `venv\Scripts\activate`

## Создание суперпользователя

```bash
manage.bat createsuperuser
```

## Создание шаблонов портфолио

```bash
manage.bat create_templates
```

## Проблемы?

Если видите ошибку "No module named 'django'":
1. Убедитесь, что виртуальное окружение создано: проверьте наличие папки `venv`
2. Если папки нет, запустите `setup.bat`
3. Всегда используйте `manage.bat` вместо прямого запуска `manage.py`
