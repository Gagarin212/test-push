========================================
ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ АДМИН-ПАНЕЛИ
========================================

1. СОЗДАНИЕ СУПЕРПОЛЬЗОВАТЕЛЯ
   ---------------------------
   
   Вариант 1 (рекомендуется):
   - Запустите: create_superuser.bat
   - Или: python create_superuser.py
   - Введите email, пароль и другие данные
   
   Вариант 2:
   - Запустите: python manage.py createsuperuser
   - Следуйте инструкциям

2. ЗАПУСК СЕРВЕРА
   --------------
   
   python manage.py runserver

3. ВХОД В АДМИН-ПАНЕЛЬ
   -------------------
   
   Откройте в браузере: http://localhost:8000/admin/
   Войдите используя email и пароль суперпользователя

4. ЧТО МОЖНО ДЕЛАТЬ В АДМИН-ПАНЕЛИ
   --------------------------------
   
   ✓ Управление пользователями
     - Просмотр всех пользователей
     - Редактирование данных пользователей
     - Управление правами доступа
     - Просмотр аватаров
     - Переход к портфолио пользователя
   
   ✓ Управление портфолио
     - Просмотр всех портфолио
     - Редактирование данных портфолио
     - Добавление/редактирование работ прямо в форме портфолио
     - Управление контактами и социальными сетями
     - Настройка дизайна
     - Просмотр статистики (количество работ)
   
   ✓ Управление работами
     - Просмотр всех работ
     - Редактирование работ
     - Изменение порядка работ прямо в списке
     - Фильтрация по типу контента, портфолио, датам
     - Поиск по названию, описанию, тегам
   
   ✓ Управление шаблонами
     - Создание и редактирование шаблонов
     - Управление активностью шаблонов
     - Загрузка превью изображений

5. ПОЛЕЗНЫЕ ФИЛЬТРЫ И ПОИСК
   ------------------------
   
   - Используйте фильтры для быстрого поиска
   - Поиск работает по множеству полей
   - Можно комбинировать фильтры и поиск

6. ПРЕВЬЮ ИЗОБРАЖЕНИЙ
   -------------------
   
   - Все изображения отображаются с превью
   - Аватары показываются с закругленными углами
   - Можно кликнуть на превью для просмотра

7. INLINE РЕДАКТИРОВАНИЕ
   ----------------------
   
   - В форме портфолио можно добавлять работы
   - Не нужно переключаться между страницами
   - Удобно для быстрого редактирования

========================================
ПРИМЕЧАНИЯ
========================================

- Только суперпользователи и персонал имеют доступ
- Все изменения сохраняются автоматически
- Используйте фильтры для работы с большим количеством данных
- Рекомендуется использовать HTTPS в production
